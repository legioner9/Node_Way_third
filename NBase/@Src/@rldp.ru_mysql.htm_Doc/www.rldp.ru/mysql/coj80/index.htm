Руководство разработчика MySQL Connector/J 8.0

Обзор

Это руководство описывает, как установить, настроить и разрабатывать приложения базы данных, используя MySQL Connector/J 8.0, JDBC и драйвер X DevAPI для связи с серверами MySQL.

MySQL Connector/J 8.0 настоятельно рекомендован для использования с MySQL Server 8.0, 5.7 и 5.6. Пожалуйста, обновитесь до MySQL Connector/J 8.0.

Для примечаний, детализирующих изменения в каждом выпуске Connector/J 8.0, см. MySQL Connector/J 8.0 Release Notes.

Для юридической информации, включая лицензирование информации, посмотрите Preface and Legal Notices.

Для помощи с использованием MySQL, пожалуйста, посетите MySQL Forums, где можно обсудить вопросы с другими пользователями MySQL.
