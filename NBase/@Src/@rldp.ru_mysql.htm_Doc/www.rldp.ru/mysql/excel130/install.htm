Глава 2. Установка

MySQL for Excel это 32-bit add-in для Microsoft Excel, который можно установить с MySQL Installer for Windows. MySQL for Excel не работает в OS X или macOS.

MySQL for Excel может взаимодействовать с MySQL Workbench, чтобы упростить управление подключениями MySQL, когда оба инструмента клиента MySQL устанавливаются. MySQL Workbench не требуется, чтобы выполнять операции с базой данных из MySQL for Excel.

Требования MySQL for Excel

  • .NET Framework 4.5.2 (Client или Full Profile).

  • Microsoft Office Excel 2007 или выше (для Microsoft Windows).

  • Visual Studio 2010 Tools for Office Runtime. MySQL Installer может установить это для вас.

    Это требование иное, чем у Office Developer Tools for Visual Studio, что не является заменой.

  • Доступная связь сервера MySQL.

MySQL for Excel загрузка и установка

Есть два варианта загрузки, доступные для установки MySQL for Excel:

  • Автономный: Загрузите и выполните файл MySQL for Excel MSI. С этим выбором необходимо гарантировать, что все требования MySQL for Excel выполнены прежде, чем выполнить файл MSI.

  • MySQL Installer: Загрузите и выполните MySQL Installer MSI. Этот выбор рекомендуется, потому что в дополнение к управлению всеми клиентскими приложениями MySQL для Windows, MySQL Installer проверяет требования к установке на вашем компьютере.

    • Когда MySQL Installer устанавливается впервые: Используйте тип установки Developer, Client Only или Full, чтобы установить MySQL for Excel вместе с другими приложениями MySQL. Используйте тип установки Custom, чтобы установить только MySQL for Excel.

    • Когда MySQL Installer уже есть: Используйте операцию Add, чтобы установить MySQL for Excel.

    Для дополнительных инструкций по установке и использованию см. MySQL Installer for Windows.

Открытие MySQL for Excel

MySQL for Excel загружается и выполняется, выбирая вкладку меню Data в Excel, затем надо нажать кнопку MySQL for Excel. Эта кнопка открывает панель задач MySQL for Excel, из которой можно выбрать опции и операции. Первоначально MySQL for Excel показывает информацию о связи, как показано в следующем примере.

Если пункты в панели задач накладываются, вы, возможно, должны перезапустить Microsoft Windows после установки MySQL for Excel.
