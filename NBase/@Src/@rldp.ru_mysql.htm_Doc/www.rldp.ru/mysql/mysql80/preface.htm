Предисловие и официальные уведомления

Это руководство для MySQL Database System 8.0. Различия между незначительными версиями MySQL 8.0 отмечены в существующем тексте в отношении номеров выпуска (8.0.x). Для информации о лицензии см. Официальные уведомления.

Это руководство не предназначено для использования с более старыми версиями программного обеспечения MySQL из-за многих функциональных и других различий между MySQL 8.0 и предыдущими версиями. Если Вы используете более ранний выпуск программного обеспечения MySQL, пожалуйста, обратитесь к соответствующему руководству. Например, MySQL 5.7 Reference Manual покрывает ряд версий 5.7 MySQL.

Лицензирование MySQL 8.0. Этот продукт может включать имеющее отношение к третьей стороне программное обеспечение, используемое в соответствии с лицензией. Если Вы используете выпуск Community MySQL 8.0, см. здесь для лицензирования информации, включая лицензирование информации, касающейся имеющего отношение к третьей стороне программного обеспечения, которое может быть включено в этот выпуск Community.
