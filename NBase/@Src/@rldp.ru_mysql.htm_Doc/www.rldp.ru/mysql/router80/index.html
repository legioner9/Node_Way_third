MySQL Router 8.0

Резюме

MySQL Router это часть кластера InnoDB и является легким промежуточным программным обеспечением, которое обеспечивает прозрачное перенаправление между приложением и MySQL Server. Это может использоваться для большого разнообразия вариантов использования, таких как обеспечение высокой доступности и масштабируемости, эффективного управления трафиком базы данных. Расширяемая архитектура также позволяет разработчикам расширить MySQL Router для разных вариантов использования. Для дополнительных деталей о том, как MySQL Router работает с кластером InnoDB, см. InnoDB Cluster.

MySQL Router 8.0 настоятельно рекомендован для использования с MySQL Server 8.0 и 5.7.

Для примечаний, детализирующих изменения в каждом выпуске, см. MySQL Router Release Notes.

Если вы еще не установили MySQL Router, загрузите его отсюда.

Для помощи с использованием MySQL, пожалуйста, посетите MySQL Forums или MySQL Mailing Lists, где можно обсудить вопросы с другими пользователями MySQL.
