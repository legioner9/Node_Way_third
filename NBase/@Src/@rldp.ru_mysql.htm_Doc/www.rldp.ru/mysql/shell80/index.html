MySQL Shell 8.0 (часть MySQL 8.0)

Обзор

MySQL Shell это продвинутый клиент и редактор кода для MySQL Server. Этот документ описывает базовые функции MySQL Shell. В дополнение к функциональности SQL, подобной mysql , MySQL Shell обеспечивает возможности скриптов для JavaScript и Python и включает API для работы с MySQL. X DevAPI позволяет вам работать с данными о документе, см. Using MySQL as a Document Store. AdminAPI позволяет вам работать с класетром InnoDB, см. InnoDB Cluster.

MySQL Shell 8.0 рекомендован для использования с Server 8.0 и 5.7. Лучше обновить до MySQL Shell 8.0. Если вы еще не установили MySQL Shell, загрузите его с официального сайта разработчиков.

Для примечаний, детализирующих изменения в каждом выпуске, см. MySQL Shell Release Notes.

Для помощи с использованием MySQL, пожалуйста, посетите MySQL Forums или MySQL Mailing Lists, где можно обсудить вопросы с другими пользователями MySQL.
