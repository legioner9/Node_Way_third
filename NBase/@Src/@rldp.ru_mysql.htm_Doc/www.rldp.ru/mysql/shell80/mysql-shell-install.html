Глава 2. Установка MySQL Shell

Эта секция описывает, как загрузить, установить и запустить MySQL Shell, который является интерактивным интерфейсом JavaScript, Python или SQL и управления MySQL Server. MySQL Shell это компонент, который можно установить отдельно.

MySQL Shell понимает X Protocol и позволяет вам использовать X DevAPI в JavaScript или Python, чтобы разработать приложения, которые общаются с MySQL Server, функционирующим как хранилище документов. Для получения информации об использовании MySQL как хранилища см. здесь.

Для версий Community и Commercial MySQL Shell : Прежде, чем установить MySQL Shell, удостоверьтесь, что у вас есть Visual C++ Redistributable для Visual Studio 2015 (доступно в Microsoft Download Center) в вашей системе Windows.

Требования

MySQL Shell доступен на Microsoft Windows, Linux и macOS для 64-битных платформ.

2.1. Установка MySQL Shell в Microsoft Windows

Чтобы установить MySQL Shell в Microsoft Windows, используя MSI Installer:

  1. Скачайте пакет Windows (x86, 64-bit), MSI Installer с http://dev.mysql.com/downloads/shell/.

  2. По запросу нажмите Run.

  3. Выполните шаги в Setup Wizard.

2.2. Установка MySQL Shell в Linux

Инсталляционные пакеты для MySQL Shell доступны только для ограниченного количества дистрибутивов Linux и только для 64-битных систем.

Для поддержанных дистрибутивов Linux самый легкий способ установить MySQL Shell в Linux состоит в том, чтобы использовать MySQL APT repository или MySQL Yum repository. Для систем, не использующих репозитарии MySQL, MySQL Shell может также быть загружен и установлен непосредственно.

    2.3. Установка MySQL Shell в OS X

    Чтобы установить MySQL Shell в OS X, надо:

    1. Загрузите пакет с http://dev.mysql.com/downloads/shell/.

    2. Дважды щелкните по загруженному DMG, чтобы смонтировать его. Должен открыться Finder.

    3. Дважды щелкните по файлу .pkg в окне Finder.

    4. Выполните шаги в инсталляционном мастере.

    5. Когда инсталлятор закончит, отмонтируйте DMG. Это может быть удалено.

    2.4. Запуск MySQL Shell

    Вам нужны имя учетной записи и пароль, чтобы установить сессию, используя MySQL Shell. Замените user вашим именем учетной записи.

    Откройте окно терминала (командная строка в Windows) и следующей командой запустите MySQL Shell: 

    mysqlsh --uri user@localhost

    Надо ввести свой пароль. Обратите внимание на то, что максимальная длина пароля, которая принята для соединения с MySQL Shell, является 128 символов.
