Глава 2. Установка MySQL for Visual Studio

MySQL for Visual Studio это add-on для Microsoft Visual Studio, который упрощает разработку запросов, используя данные из MySQL. Многие особенности MySQL for Visual Studio также требуют, чтобы MySQL Connector/NET был установлен на том же самом хосте, где вы выполняете разработку в Visual Studio. Connector/NET это отдельный продукт с несколькими версиями.

Возможности для установки MySQL for Visual Studio:

  • MySQL Installer (лучше): скачайте и выполните MySQL Installer.

    С этим выбором можно загрузить и установить MySQL Server, MySQL for Visual Studio и Connector/NET из того же самого пакета программного обеспечения, на основе версии сервера. Первоначально MySQL Installer помогает вам, оценивая предпосылки программного обеспечения, необходимые для установки. После этого MySQL Installer позволяет вам поддерживать свои установленные продукты обновленными или легко добавить и удалить связанные продукты MySQL.

    Для получения дополнительной информации об использовании MySQL Installer см. MySQL Installer for Windows.

  • Используя автономный файл Zip или MSI: этот выбор идеален, если у вас уже есть MySQL Server и Connector/NET. Используйте информацию в этом разделе, чтобы определить, которая версия MySQL for Visual Studio установлена.

MySQL for Visual Studio Configuration Update Tool

MySQL for Visual Studio обновляет свою конфигурацию автоматически, когда Connector/NET устанавливается на вашем компьютере. Однако, если вы устанавливаете Connector/NET после MySQL for Visual Studio или обновляете Connector/NET после установки MySQL for Visual Studio, будет запрос на запуск Configuration Update Tool.

Отклонение конфигурации происходит, если найден MySQL for Visual Studio, когда вы пытаетесь создать модель Entity Framework или в первый раз, когда вы пытаетесь открыть, создать или отредактировать связь с MySQL. Произвольно MySQL for Visual Studio может обновить конфигурационные файлы для вас, когда это обнаруживает ошибки конфигурации и обеспечивает всплывающее окно, побуждающее вас вызвать инструмент. Выберите Yes, чтобы управлять инструментом. Необходимо перезапустить Visual Studio после того, как затронутые конфигурационные файлы будут обновлены.

Минимальные требования

MySQL for Visual Studio работает с несколькими версиями Visual Studio, хотя степень поддержки основана на ваших установленных версиях Connector/NET и Visual Studio. MySQL for Visual Studio 1.2.9 больше не поддерживает Visual Studio 2013 или 2012.

Минимальные требования для поддерживаемых версий Visual Studio следующие:

  • Visual Studio 2019 (Community, Professional и Enterprise).

    MySQL for Visual Studio 1.2.9 с Connector/NET 8.0.14.

    Рекомендуется Connector/NET 8.0.18.

  • Visual Studio 2017 (Community, Professional и Enterprise):

    MySQL for Visual Studio 1.2.7 or 2.0.5 с Connector/NET 6.9.8.

  • Visual Studio 2015 (Community, Professional и Enterprise):

    MySQL for Visual Studio 1.2.7 или 2.0.2 с Connector/NET 6.9.8.

Минимальные требования для ранее поддерживаемых версий Visual Studio включают:

  • Visual Studio 2013 (Professional, Premium, Ultimate):

    • .NET Framework 4.5.2 (установлен первым).

    • MySQL for Visual Studio 1.2.1 или 2.0.0 с Connector/NET 6.9.8.

  • Visual Studio 2012 (Professional, Test Professional, Premium, Ultimate):

    • .NET Framework 4.5.2 (установлен первым).

    • MySQL for Visual Studio 1.2.1 или 2.0.0 с Connector/NET 6.9.8.

MySQL for Visual Studio не поддерживает версии продуктов разработки Microsoft, включая Visual Studio и Microsoft Visual Web Developer. Чтобы использовать Connector/NET с версиями Express, применяйте Connector/NET 6.9 или позже, не устанавливая MySQL для Visual Studio.

Ограничения MySQL Connector/NET

MySQL for Visual Studio тесно связывается с Connector/NET, но они два отдельных продукта, которые могут использоваться друг без друга. Следующие ограничения применяются:

  • MySQL for Visual Studio не может быть установлен вместе ни с какой версией Connector/NET 6.6 и ранее, который должен быть удален прежде, чем установить MySQL for Visual Studio.

  • Следующие особенности MySQL for Visual Studio требуют Connector/NET:

    • Entity Framework Designer.

    • Application Configuration tool.

    • Отладка хранимых процедур и функций.

    • DDL T4 Template Macro (чтобы произвести базу данных из модели EF).
