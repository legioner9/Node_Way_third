MySQL Workbench

Обзор

Это справочник по MySQL Workbench. Это документирует MySQL Workbench Community и MySQL Workbench Commercial Editions 8.0 по 8.0.15.

Если вы еще не установили MySQL Workbench Community Edition, пожалуйста, загрузите свой бесплатный экземпляр с download site. MySQL Workbench Community Edition доступен для Microsoft Windows, macOS и Linux.

Для примечаний, детализирующих изменения в каждом выпуске, см. MySQL Workbench Release Notes.

Для юридической информации см. Legal Notices.

Для помощи с использованием MySQL, пожалуйста, посетите MySQL Forums или MySQL Mailing Lists, где можно обсудить вопросы с другими пользователями MySQL.

If you are using MySQL Workbench Commercial Editions, see the MySQL Workbench Commercial License Information User Manual for licensing information, including licensing information relating to third-party software that may be included in this Commercial Edition release. If you are using the MySQL Workbench Community Edition, see the MySQL Workbench Community License Information User Manual for licensing information, including licensing information relating to third-party software that may be included in this Community Edition release.
