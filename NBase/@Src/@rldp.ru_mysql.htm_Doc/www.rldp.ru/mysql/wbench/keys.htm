Приложение B. Клавиатурные сокращения

Следующие таблицы приводят клавиши быстрого вызова для команд MySQL Workbench. Modifier обозначает определенную для платформы клавишу модификатора. Это Command в macOS, Control в других платформах. На macOS Alt это Option.

Есть клавиши быстрого вызова для меню File, Edit, View, Arrange, Model, Query, Database, Scripting, Help и EER Diagram Mode.
