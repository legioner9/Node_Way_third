Блоги на Sql.ru

Открыт новый раздел "Блоги" на Sql.ru.

Мы приглашаем всех желающих завести свой новый "технический" блог на Sql.ru или транслировать существующий.
Основным преимуществом заведения блога, является доступ к большой специализированной аудитории, что позволит привлечь новых читателей и получать больше отзывов на публикуемые сообщения.
Опубликованные сообщения, будут также доступны для чтения в других тематических разделах сайта, в соответствии с выбранными ключевыми словами (тегами) обеспечивая еще больше читателей.
