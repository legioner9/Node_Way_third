Динамический SQL в программах на PL/SQL. Автор: Steve Bobrovsky
     ORACLE     
29-03-2001		 Динамический SQL в программах на PL/SQL. Использование пакета DBMS_SQL. Администраторы и разработчики часто пренебрегают теми преимуществами, которые можно получить от применения некоторых выдающихся, но скрытых драгоценных возможностей, которыми обладает сервер Oracle7. В Oracle7 версии 7.1 имеется поистине фантастическое добавление к семейству пакетов DBMS: новый пакет DBMS_SQL обеспечивает разработчиков инструментарием для создания динамически формируемых предложений SQL в программах на PL/SQL.  [почитать]

RCO for Oracle RCO
     ORACLE     
01-11-2002		 Продукт позволяет существенно расширить возможности СУБД Oracle при работе с русскоязычными базами данных, обеспечивая анализ и поиск текстовой информации с учетом особенностей русского языка. RCO for Oracle задействует такие технологии, как лексико-грамматический и статистический анализ текста; алгоритмы автоматической классификации, рубрицирования и реферирования; нечеткий поиск и др. Существуют версии продукта для Windows и различных UNIX платформ (SUN Solaris, SCO UnixWare, Compaq Tru64 Unix, HP UX, IBM AIX).  [почитать]
